Building Forms in React has never been faster

React Form is an extensive, simple and efficient solution for creating basic to complex forms in React. Out of the box you get the ability to grab and manipulate values, set validation (errors, warnings and successes), customize your inputs, perform asynchronous validation and much more!

Basic Setup

Hop into your existing React project and install React Form. New to React? Don't fret! 

npm i react-form
